### Gerador de insulto

```
sério
enfia
isso aí
no rabo
porra

tá loco
```

Abra a [demo](https://vitormanfredini.github.io/gerador-de-insulto/) e recite em voz alta para reduzir o estresse. Atualize a página para um novo insulto.
